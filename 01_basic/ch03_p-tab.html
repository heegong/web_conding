p 태그

문서의 단락을 표시 하기 위해서 사용하는 태그입니다. 한줄 안에 표시가 안되는 내용은 자동으로 줄바꿈해서 표시해줍니다.

이전 단락과 구분되는 새로운 단락으로, 단락과 단락 사이에 공간이 형성됩니다.

줄바꿈이 필요한 경우는 p태그를 사용할 수 도 있지만 br태그를 사용하면 더 편하다. br은 닫는 태그가 없다.


특수 기호

: 웹 브라우저에 기호를 표시 할때 '&'기호를 사용해서 작성합니다.

& : amp 앤드기호

  : nbsp 공백         백

< : lt 꺽쇠 괄호 열기

> : gt 꺽쇠 괄호 닫기
